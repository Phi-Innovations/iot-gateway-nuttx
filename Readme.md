# PHI-Gateway

This is a PHI Gateway firmware done in C++ running on Nuttx.

# Features

* Use Nuttx as a library (nuttx-export-10.0.1)
* Use Cmake as build engine
* Has automation to load firmware on the board using Segger J-Link

# Build instruction

```
$ make default
```

# Load the firmware

```
$ make run
```
